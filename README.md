# Image-Handling-and-Pixel-Transformations-Using-OpenCV-
# AIM:
To develop a Python program using OpenCV for performing image handling and pixel transformation operations such as image reading, displaying, brightness adjustment, contrast enhancement, cropping, resizing, flipping, channel splitting, merging, and bitwise operations.

# Software Required:
Anaconda – Python 3.7

Jupyter Notebook

OpenCV

Python Libraries: cv2, numpy, matplotlib

# Algorithm:
## Step 1:
Import the required libraries such as OpenCV, NumPy, and Matplotlib.

## Step 2:
Read the image using cv2.imread() in grayscale and color modes.

## Step 3:
Display the image using matplotlib.pyplot.imshow().

## Step 4:
Find and print the image properties such as width, height, and number of channels.

## Step 5:
Save the processed image using cv2.imwrite().

## Step 6:
Crop a specific portion of the image and resize it by a factor of 2.

## Step 7:
Flip the cropped image horizontally using cv2.flip().

## Step 8:
Read the Apollo launch image and annotate it by adding text and drawing rectangles.

## Step 9:
Adjust image brightness by adding and subtracting a matrix of ones using cv2.add() and cv2.subtract().

## Step 10:
Modify image contrast using scaling factors such as 1.1 and 1.2.

## Step 11:
Split the image into B, G, R channels and display each channel separately.

## Step 12:
Merge the RGB channels back into the original image.

## Step 13:
Convert the image into HSV format and split it into H, S, V channels.

## Step 14:
Merge the HSV channels and display the final image.
# Concept Overview
## 1. OpenCV
Open-source computer vision library. Used for image and video processing.

## 2.Brightness Adjustment
Changes pixel intensity. Makes image brighter or darker.

## 3.Contrast Adjustment
Changes difference between light and dark areas. Improves image clarity.

## 4.RGB Color Model
Consists of Red, Green, and Blue channels. Used in digital images.

## 5.HSV Color Model
Represents Hue, Saturation, and Value. Useful for color detection.

## 6.Bitwise Operations
Includes AND, OR, XOR, and NOT operations. Used for masking and image blending.

## 7.Applications of Image Processing
Medical imaging. Facial recognition. Satellite imaging. Artificial intelligence applications.
